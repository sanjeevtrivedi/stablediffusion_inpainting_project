# stablediffusion_inpainting_project

Topic: Inpainting using Diffusion models
Task Description: Use diffusion models to reconstruct missing or corrupted regions in image.
Deliverables: Dataset preparation, understanding of inpainting technique, experimentation, and evaluation using PSNR, SSIM and visual quality.
Detailed Task Description: Expected Deliverables: Understand the concepts, do literature survey, Prepare script for inpainting using diffusion model, Suitable dataset (avoid large datasets to avoid large scale training) and evaluation metrics, Experimentation with result analysis, Insights of the result and visualization.
